# AI-Enhanced UI Prototypes

A collection of AI-enhanced UI patterns and prototypes exploring the intersection of Generative UI, CopilotKit, and the Vercel AI SDK. Built with Next.js 15, React 19, and Tailwind CSS v4.

## Demos

### 1. Sparkle Enhancement (`/sparkle-demo`)
Simple textarea with AI enhancement button. Click the sparkle icon to polish your text using GPT-4o-mini with real-time streaming.
- **Tech:** Vercel AI SDK, `useCompletion`, streaming

### 2. Multi-Step Wizard (`/profile`)
4-step profile wizard with AI sparkle enhancement on every field. Demonstrates the sparkle pattern across a multi-step form flow.
- **Tech:** Vercel AI SDK, multi-step forms

### 3. CopilotKit Basic (`/copilot-test`)
Introduction to CopilotKit with `useCopilotReadable` and `useCopilotAction`. Shows basic Generative UI patterns.
- **Tech:** CopilotKit, Generative UI

### 4. AI-Powered Wizard (`/ai-wizard`)
Full wizard with chat-driven auto-fill, field enhancement, and step navigation. The AI can fill forms, enhance fields, and navigate between steps.
- **Tech:** CopilotKit, Generative UI, Forms

### 5. Interactive Onboarding (`/ai-onboarding`)
Conversational onboarding with clickable Generative UI cards. Split-screen layout with form on left and chat on right.
- **Tech:** CopilotKit, Generative UI, Split-screen

### 6. Hybrid Sparkle + Chat (`/hybrid-demo`)
Best of both worlds: Quick sparkle buttons for single fields + chat sidebar for bulk operations and context-aware assistance.
- **Tech:** Vercel AI SDK + CopilotKit hybrid

### 7. AI Dashboard Assistant (`/ai-dashboard`)
SaaS metrics dashboard with AI insights. Ask questions about your data, get explanations, detect anomalies, and generate reports.
- **Tech:** CopilotKit, Recharts, Data Visualization

### 8. AI Compare & Decide (`/ai-compare`)
Comparison tables, pros/cons lists, recommendations, and decision matrices generated by AI. Great for product comparisons and decision-making.
- **Tech:** CopilotKit, Generative UI

### 9. Sliding Chat Panel (`/ai-sidepanel`)
Click the floating chat icon to open a slide-out side panel. Business document form with AI assistance that can fill entire documents.
- **Tech:** CopilotKit, Side Panel, Forms

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **React:** React 19
- **Styling:** Tailwind CSS v4 + Dark Mode
- **AI SDK:** Vercel AI SDK (`useCompletion`, `streamText`)
- **Copilot:** CopilotKit (Generative UI, Actions, Readable)
- **Charts:** Recharts
- **Icons:** Lucide React
- **LLM:** OpenAI GPT-4o-mini

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- OpenAI API key
- CopilotKit Cloud API key (optional, for CopilotKit demos)

### Environment Variables

Create a `.env.local` file:

```bash
OPENAI_API_KEY=your_openai_api_key_here
NEXT_PUBLIC_COPILOT_CLOUD_API_KEY=your_copilotkit_cloud_api_key_here
```

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the demo gallery.

## Key Patterns

### Sparkle Enhancement Pattern
Quick, inline AI enhancement with a button click. Uses Vercel AI SDK for streaming responses.

### Generative UI Pattern
AI actions that render React components directly in the chat. The AI decides when to show cards, tables, or interactive elements.

### Split-Screen Layout
Form/content on the left, chat on the right. Great for guided workflows.

### Sliding Panel Pattern
Floating chat button that opens a slide-out panel. Non-intrusive AI assistance.
